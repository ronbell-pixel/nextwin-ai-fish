# NextWin AI Website

A static, responsive website for NextWin AI.

## Files
- `index.html` - Main website
- `styles.css` - Site styling
- `script.js` - ROI calculator logic
- `assets/logo.svg` - Simplified NextWin AI logo
- `assets/favicon.svg` - Browser favicon

## How to preview
Open `index.html` in your browser.

## How to deploy on Cloudflare Pages
1. Create a new GitHub repository.
2. Upload these files to the repository root.
3. In Cloudflare, go to Pages > Create a project.
4. Connect the GitHub repo.
5. Build command: leave blank.
6. Output directory: `/` or leave default.
7. Deploy.

## Items to customize before launch
- Demo phone number is currently set to `314-406-8800`.
- Replace `hello@nextwinai.com` with your email.
- Replace the contact form with My AI Front Desk, GoHighLevel, HubSpot, Calendly, or your CRM form.
- Add legal pages: Privacy Policy, Terms, AI Disclosure.
